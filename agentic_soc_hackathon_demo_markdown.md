# 🛡️ Agentic SOC – Hackathon Demo (Web + AI Senaryo)

## 🚀 Amaç
Bu doküman, **yarınki hackathon için hızlı ve etkileyici bir demo sunumu** oluşturmak amacıyla hazırlanmıştır.

Hedef:
- 5-10 dakikalık demo
- Görsel olarak güçlü dashboard
- "Agentic AI çalışıyor" hissi
- Teknik + iş değeri birlikte anlatım

---

# 🧠 Demo Hikayesi (Storyline)

## 🎬 Senaryo

> "Bir kamu kurumuna saldırı oluyor ve sistem bunu otomatik tespit edip aksiyon alıyor"

Akış:
1. Sahte bir zararlı dosya sisteme girer
2. Sistem bunu analiz eder
3. Agent’lar devreye girer
4. Karar verilir
5. Dashboard’da gösterilir

# 🌐 Web Demo Yapısı

## 🖥️ 1. Ana Dashboard

### Gösterilecek Paneller:

#### 🔴 Threat Overview
- Aktif tehdit sayısı
- Kritik / Orta / Düşük dağılım
- Gerçek zamanlı animasyon (fake olabilir)

#### 📊 System Status
- LLM: Online
- Agents: Running
- Sandbox: Active

#### ⚡ Live Alerts
- "Suspicious IP detected"
- "Malware behavior observed"

---

## 🔍 2. Threat Detail Panel

Bir karta tıklanınca açılır:

- Dosya adı
- Risk skoru
- Kaynak IP
- Davranış analizi

---

## 🤖 3. Agent Activity Panel

Burada "Agentic AI" hissi verilecek:

Örnek loglar:

- "LLM analyzing file..."
- "Sandbox execution started"
- "Behavior anomaly detected"
- "Legal agent generating report"

---

## 🧪 4. Sandbox Görselleştirme

Fake ama etkili:

5. Sonuç üretilir
